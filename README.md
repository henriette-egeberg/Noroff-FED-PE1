# Noroff-FED-PE1

My 2025 Exam Project, where I was tasked with creating a Ecommerce website for a fictional Client.
Time Commitment: 10 Weeks
Terminology: 
  - User: a person visiting the sithe who is not logged in
  - Owner: the logged-in manager of the shop

## Features

- View products (Description, Price, Rating, and Reviews when Available)
- Log In to to purchase products
- Log Out after checking out

## Prerequisites

- Node.js (v20+)
- npm

## Getting Started

This website is supposed to mimick a real life Ecommerce. Payment etc will not work, only simulate.

### Installation

```bash
npm install
```

### Running the project

```bash
npm run dev
```

### Running tests

```bash
npm run test
```

## Environment Variables

Create a `.env` file in the root directory:

```bash
API_KEY=your-api-key-here
BASE_URL=https://example.com/api
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run test` - Run tests
- `npm run lint` - Run ESLint
